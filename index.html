我的Mac常用开发APP

Git

  • SourceTree 脱离命令行用git的最佳途径
  • iterm2 可以代替自带终端的神器，字体高亮，各种DIY没问题
  • tig 一个命令行插件，可以在命令行看到git的 commit log

网络请求

  • Postman 用来模拟请求，自带的Pre-request Script可以用来请求前写一些脚本（md5参数什么的），还能跑测试，非常强大，有chrome插件版和客户端（其实就是封装了个插件进客户端）
  • PAW 也是用来模拟请求的，界面更加符合mac设计，收费
  • Surge 大部分用来科学上网，但是实际上也能够捉包测速！非常有用！以前如果买过iOS版的直接附送一份mac版 收费
  • 花瓶 经典的捉包工具，收费

编程

  • Sublime Text 编写代码通用神器，这个不用说了，不过平时用来临时看看代码和写些小脚本用
  • Xcode 好多bug但是又不能不用的
  • atom 用这个来写react-native程序，插件也很丰富
  • mweb 我用这个来写这篇文章的，能够一键发布blog，很好用，收费
  • MacDown 比较轻量的用来写markdown的软件，不过有了mweb就可以代替这个了
  • Linguan 可以同时处理中英文的国际化文件，不需要像Xcode那样切换两个文件来处理 收费

图像

  • Photoshop CC 搞设计图的
  • cutterman ps的插件，如果设计师懒的话，开发的通过这个插件来从psd中抠图（很好地支持iOS和Android）
  • Sketch 这个完全可以代替Photoshop，不过这个世界上实际太多人用Photoshop了，需要走很长的路，性能和界面完秒Photoshop，对应社区也有很丰富的插件和生态，未来的趋势
  • Pixel Winch 可以测量设计图间距，支持自动测距，有些误差，不过也算比较高效率了，收费，而且真特么贵
  • Gapplin 可以打开和导出SVG格式的图片
  • ImageAlpha 用来压缩PNG，可以自定义压缩率，效果不错，不过一次只会处理一张
  • TinyPNG WEB版的压缩PNG工具，可以一次上传多张图片，效率高，但是不能够自定义压缩率
  • Prepo 提供一个1024*1024的jpg就可以压缩所有设备所需要的icon了

iOS发版本相关

  • iPhone配置使用工具 可以很方便地处理所有provision file，而不需要在Xcode搞，不过据说Xcode8也优化了provision的引用方式
  • Apple Configuration 开发者可以方便将ipa安装到设备上面，而不需要通过iTunes来同步之类的
  • Application Loader 上传app到AppStore，比Xcode的成功率高一些

我的日常git

解决 git pull 冲突

  • 冲突了某文件（或者文件夹）
#用Head的文件去覆盖自己的文件
git checkout --theirs file1 file2 ... 
#覆盖文件夹
git checkout --theirs dir1/. dir2/.

#使用自己的文件
git checkout --ours file1 file2 ...
#覆盖文件夹
git checkout --ours dir1/. dir2/.

Archive with CLI

使用几种方式去对项目进行打包操作

使用shenzhen

先clean并且根据PP文件（这里是Ad Hoc）打包

# -w workspace位置名称
# -s schema名称
# -c config名称
ipa build -w "qingxun.xcworkspace" -s "qingxun" -c "Ad Hoc" --clean
# 上传到itunesConnect
ipa distribute:itunesconnect -a dev@tupu.im -p Tupu12345 -i 915233164 --upload

打包Demo（clean - 打包 - 上传到蒲公英）

#!/bin/sh

ipa build -w "liangce.xcworkspace" -s "liangce" -c "Ad Hoc" --clean

unzip liangce.app.dSYM.zip
leancloud upload_symbol -v \
-f "./liangce.app.dSYM" \
-i "AsXGHqesNo0paA0qQdW0anXt-gzGzoHsz" \
-k "6NBEasX2Rl0487o7Lmolxtzq"

rm -rf liangce.app.dSYM
ipa distribute:pgyer -u 6f96c00c6382e5109c539246bea00e38 -a b80c23ea536df7224b2d71eb6c1df022
rm -rf liangce.ipa
rm -rf liangce.app.dSYM.zip

使用XcodeBuild 命令打包

#!/bin/sh
project=qingxun  #Project name
configuration=Ad\ Hoc   #Build configuration
xcodebuild clean -workspace $project.xcworkspace -scheme $project
xcodebuild archive -workspace $project.xcworkspace -scheme $project -configuration $configuration -archivePath $project.xcarchive
xcodebuild -exportArchive -archivePath $project.xcarchive -exportPath $project -exportFormat ipa

使用Gym打包

#!/bin/sh
project=qingxun
gym --workspace "$project.xcworkspace" --scheme "$project" --clean --use_legacy_build_api
rm -rf $project.app.dSYM.zip
